# ホーム画面について
1. ホーム画面でお二人の式場探しと書いてあるが、”し”だけが改行されてて見栄えが悪い
2. 比較とか追加とか見学とか、押せそうに見えるけど押せないボタンが並んでてなんだよってなる
3. 次の一歩で、候補を並べてふたりで比べてみましょうのところで比べるって押すとうまくいきませんでした、という画面が候補画面に遷移したのちに出てくる。そこでもう一度試すを押しても同じ、というかそもそも一切反応がない。
4. 比較してみましょうみたいなガイドが出てるけど次の一歩と同じこと言ってて冗長に感じる。あと実際に比較するを押しても３番と同じことが起こる。
5. 比較するって言ったり比べるって言ったり表現が揺れてる
6. レイアウトデザインが全くラグジュアリーでも美しくもない、最大のポイントは全くモダンじゃない。２０年前のデザインみたい。２０２６年４月最新の技術力をふんだんに使ったモダンでユーザーを感動させるビジュアルデザインにしてほしい
7. 最近ご覧になった式場で、すべてを押すと探すのページにたどり着くのちょっと意味がよくわからない。一覧画面と本当に探すための画面と追加するための画面と別れるのが正しいのかよくわからないけど、いずれにしても書いてることとそこで行える機能と起きることが合致してなくて気持ち悪く感じる。画面一覧、情報設計、機能配置設計、遷移設計をUIUX最大化の観点で考え直すべき

# 探す画面について
8. 追加ボタンが目立たな過ぎてUX体験が悪い気がする
9. 探す画面って書いてるのに気になる式場を集める場所って書かれてるのも気持ち悪い
10. 新しい式場の追加でURLを張ると、AI解析に失敗しました。手動で入力してください。とでる。後出てる時間が少し長すぎて邪魔。画面のどっかしらタップしたら消えてほしいし逆にそのメッセージをタップし続けている間だけは表示されていてほしい。あと失敗した後に自分でのタブに戻るけど、URLのタブに移りなおしたらさっき失敗したURLがボックスの中に残ったままなのが気持ち悪い、クリアになっていてほしい
11. 新しい式場の追加でまとめてのところ、URLで張るんだったら左にURLタブがあるの気持ち悪いな。MECEというか構造化されたというか、そういう作りになってないしユーザフレンドリーでもない
12. 新しい式場を追加する画面、ぜんぜんわくわくしない。つまらないUIというかビジュアル。
13. AIおすすめ式場がずっと今少し混雑しています、少し後でまたお試しくださいの表示のまま。そもそもこれどんなところにあんねん。
14. 各式場をタップして出てくる画面だけど、まず施設名、住所、アクセスなど書かれてるところ、何の項目なのかが一定の書式で書かれてないから見づらい。あと星を更新したら評価の更新に失敗しましたと出る。後星は０．５刻みで表現できたほうがいい。あとずっと下のほうにつらつらとボタンや機能が並ぶけど、もっといいUXというか画面遷移というか動線あるんじゃ？画面切り替えというかスライドというかで。それから見積もりの項目名だけど、わりと定番なものはプリセットでプルダウンから選べて、そこにないものだけ自分で自由に入力できるほうがいい。後ほかのカップルも同程度の調整をしています、みたいなの２回出てくるのは何で？
15. 口コミのまとめだけど、気になる点を先にのボタンが何を意味するのか分からん。ナビゲートとか追加実装予定だったっけ？あと口コミの追加についてもAIの解析に失敗しましたと出る
16. 全体的にとにかく上から下へ機能とかボタンをひたすら並べましたって感じで非常に使いづらい。流れも意味も分かりにくい。直感的に操作できない。UIUXは最悪。
