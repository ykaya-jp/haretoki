この仕掛けが狙うもの

一般的なウェディングアプリが「検索 → 比較 → 予約」の機能集なのに対して、 Haretoki は 「毎朝の光のような存在」 になる。

妻がスマホを開くたび「今日のふたり」を 肯定 し、小さな一歩 を示す。 Duolingo のストリークではなく、Things 3 の "Today" でもなく、
"お茶を淹れてくれる伴走者" を目指す。

実装は Claude Sonnet 4.6 + JSON schema + Vercel Cron で 1 日 1 回生成。 プロジェクト 1,000 件で月コスト ~¥3,000 / ユーザー。
