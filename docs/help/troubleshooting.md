# トラブルシューティング

`/help` page では summary だけ表示する。詳しい手順はこちらに集約する。

## 1. ログインできない

- パスワードリセット: ログイン画面 → 「パスワードをお忘れですか？」 →
  登録メールに届くリンクを 30 分以内にタップ。
- Google ログインだけが効かない: ブラウザの third-party cookie が
  有効か確認 (Safari は ITP で 7 日後に切れることがある)。
- 招待リンクからのサインアップ後にログインできない: token 期限切れ
  (30 日)。 オーナーの方に新しい招待リンクを生成してもらう。

## 2. パートナー招待が届かない

- 受信側のメールアプリ → 迷惑メール / プロモーションタブを確認。
- それでも見当たらない場合は、 オーナーがマイページ →「パートナーを招く」
  から招待リンクを **直接コピー** して LINE / メールで送るのが確実。
- 一部の企業ドメインでは Resend からのメールがブロックされることがある。
  個人のメールアドレスへの送信を推奨。

## 3. 評価が partner 側に反映されない

- 1〜2 秒のラグは仕様 (Realtime broadcast の遅延)。
- 1 分以上反映されない: ブラウザの「最新の情報に更新」 (Cmd/Ctrl+R) を一度。
- 二人とも同じ式場画面を開いていて、 自分の評価の方が新しい場合は、
  画面下に「保存しました」 toast が出るまで他方の更新が後に来る場合がある。

## 4. 見積もり PDF が読み込まれない

- ファイルサイズ ≤ 32 MB、 形式は PDF。
- 写真ベースの PDF (スキャン画像のみ) は文字認識精度が低いため、
  見積もり JSON への自動変換が不完全になる可能性がある。 「内容を確認」
  画面で手動修正できる。
- 「読み込みに失敗しました」と出たら、 ファイル名に絵文字 / 機種依存文字
  が含まれていないか確認。

## 5. AI コーチが固まる / 返事しない

- 接続が切れている可能性。「オフラインで一時保存しました」 が出ていれば
  オフライン状態。
- 30 秒以上 spinner のまま: ブラウザを再読み込み。
- AI 提供元 (Anthropic) の障害時は、 一時的に AI 機能が縮退する。
  「AI が応答していません」 toast が出たら数分後に再試行。

## 6. リマインダー / Push 通知が届かない

- マイページ → 「通知」 → リマインダー設定でその phase が ON か。
- 端末の OS 通知設定 (Safari / Chrome) で本サイトの通知許可を確認。
- Email リマインダーが急に止まった: バウンス検知で自動停止している可能性。
  マイページ → 「メールが届きません？」 から再開できる。

## 7. 退会 / データダウンロードができない

- マイページ →「設定」→ 一番下までスクロール。
- 「記録をダウンロード」 でメール添付なしの直 download (JSON、 zip)。
- 「アカウントを削除」 でメールアドレス再入力 + 確認。 削除後は復元不可。

## 8. 別の端末で開いたら不整合

- LWW (Last Write Wins) で server 側の最新が勝つ仕様。 端末 A で書いて
  オフライン中に端末 B で書くと、 後にオンライン化した方の編集が
  最終的に server に乗る。
- 「保存待機中」 indicator が表示されている間はまだ flush されていない。
  オンラインに戻ると自動 flush。

## 9. 家族・友人共有リンクが開けない

- リンクの有効期限は 30 日。 期限切れ / オーナーが取り消した場合は 404 表示。
- リンクは公開なので、 不特定多数に共有しないこと。 オーナーの方は
  マイページから view 数を確認できる。

## 10. それでも解決しない場合

- [`/support`](#) お問い合わせフォームから、 状況・端末・ブラウザを書いて
  お送りください。 通常 3 営業日以内にお返事します。
- 緊急の場合 (アカウント乗っ取り疑い等) は件名に「緊急」 を入れてください。
