## 背景

Harenohi（旧VenueLens）は結婚式場の比較・評価・最終決定を支援するWebアプリ。
機能は Release 1-4 まで全て実装済み・デプロイ済みだが、**UI/UXのデザイン品質が低い**。
Sonnet subagent に実装を委譲した結果、「機能は動くが見た目が平凡で、ワクワクしない」状態。

**ターゲットユーザー**: 20-30代の女性（花嫁）が主要ユーザー。パートナー（男性）は招待されて使う。

## デザインの方向性

### ムードボード

| キーワード | 参考 | 避けること |
|-----------|------|-----------|
| **フェミニン × ラグジュアリー** | Aesop, Onefinestay, CLANE, Maison de FLEUR | ゴリゴリのNavy SaaS感 |
| **温かい × 洗練** | Pi AI (ベージュ+グリーン), Apple (余白+クリーン) | 冷たいダーク UI |
| **かわいい × 上品** | 日本のハイエンドブライダル（TRUNK, 八芳園） | チープなピンク |
| **自然 × 光** | 太陽・晴れの日のイメージ。暖色系の光 | 人工的・テック感 |

### カラーパレット（参考方向）

現状の Navy (#1E3A5F) を主色にするのをやめる。以下の方向に変更：

| 用途 | 現状 | 新方向 |
|------|------|--------|
| 背景 | 冷たい白 (#FAFBFC) | 温かいクリーム/ベージュ (#FAF7F2 〜 #FFF8F0) |
| メイン文字 | ダークネイビー (#0D1B2A) | ウォームダーク (#2C2420 〜 #1A1614) |
| アクセント | ネイビー (#1E3A5F) | コーラル/テラコッタ or ローズ (#C97B6B 〜 #D4856A) |
| ゴールド | そのまま活かす (#C9A84C) | やや温かめに調整 |
| カード | 純白 (#FFFFFF) | オフホワイト (#FFFCF8) |
| ボーダー | 冷たいグレー (#E0E5EB) | ウォームグレー (#E8E2DA) |

**重要**: 最終パレットは Refero で Onefinestay, Aesop, 高級ブライダルサイトを調査して決定すること。上記はあくまで方向性。

### タイポグラフィ

- 見出し: **Noto Serif JP** (weight 300-400) — これは現状維持でOK。ラグジュアリー感の源泉
- 本文: **Noto Sans JP** — 維持
- 数字: **Geist** — 維持
- ロゴ: Noto Serif JP weight 300 + 太陽アイコン

### モーション要件

| 場所 | アニメーション | ライブラリ |
|------|-------------|----------|
| ランディングヒーロー | スタガードフェードイン (各要素 0.12s 遅延) | framer-motion |
| ページ遷移 | フェード + スライドアップ (200ms) | framer-motion |
| カードホバー | translateY(-4px) + shadow拡大 (300ms) | CSS transition |
| カードタップ | scale(0.98) (150ms) | CSS active: |
| ハートタップ | scale bounce 1→1.3→1.0 (200ms) | framer-motion |
| 星評価タップ | scale(0.9) bounce (120ms) | CSS |
| リストアイテム | スタガードフェードイン | framer-motion |
| 進捗リング | 0→現在値 (600ms ease-out) | CSS transition |
| 決定セレモニー | コンフェッティ (800ms) | canvas-confetti |
| BottomNav タブ切替 | アイコン微scale + 色遷移 (150ms) | CSS |
| ボタンホバー | subtle glow / shadow拡大 | CSS |
| prefers-reduced-motion | 全て無効化 | globals.css |

## 具体的に修正すべき UI/UX 問題（ユーザー指摘の全リスト）

| # | 問題 | あるべき姿 |
